install wt

One command. Single binary, no dependencies. macOS and Linux, x64 and arm64.

macOS arm64 / x64
Linux arm64 / x64

quick install

$ curl -fsSL https://wingthing.ai/install.sh | sh

Installs to ~/.local/bin/wt. Set WT_INSTALL_DIR to change the location.

get started

  1. log in

    Authenticate with the roost so your machine is reachable from the web.

    $ wt login

  2. start your wing

    Your machine is now online. Open app.wingthing.ai to start a terminal.

    $ wt start # alias for wt wing start

  3. open a terminal

    Press . or + in the web app to open the command palette and start a session on your machine.

manual download

Grab the binary from GitHub releases and put it in your PATH.

update

$ wt update

Downloads the latest release and restarts the wing daemon if running. You can also trigger updates remotely from the web dashboard.
