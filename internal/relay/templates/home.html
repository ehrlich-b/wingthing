run sandboxed agents from anywhere

Claude Code, Codex, Ollama — sandboxed on your machine, accessible from any device. One CLI, any backend.

$ curl -fsSL https://wingthing.ai/install.sh | sh
$ wt egg claude # sandboxed Claude Code session $ wt wing -d # make your machine reachable remotely $ open app.wingthing.ai # start terminals from your browser
sandboxed Every session runs in its own sandbox. Seatbelt on macOS, namespaces on Linux. The sandbox IS the permission boundary — configure mounts, deny paths, isolation level.
remote Tailscale for agents. wt wing connects outbound — no port forwarding, no static IP, works behind any NAT. E2E encrypted. Start a terminal from your phone.
any agent Claude Code, Codex, Ollama, Cursor. Swap with a flag. wt egg ollama for free, wt egg claude for the hard stuff.
local-first Your machine, your keys, your data. Self-host the roost or use ours.
$ wt egg claude # local sandboxed session $ wt egg list # see active sessions $ wt wing status # check remote daemon $ wt doctor # detect agents, keys, services
