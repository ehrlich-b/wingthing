your ai swiss army knife

One CLI, every agentic backend. New tool drops, wt skill add it. Done.

$ go install github.com/ehrlich-b/wingthing/cmd/wt@latest
any agent Claude Code, Ollama, Gemini. Swap with a flag. --agent ollama for free, --agent claude to invoke Claude Code in a sandbox for the hard stuff.
skills Markdown files with superpowers. Curated, version-controlled. No app store, no malware marketplace.
sandboxed Skills run in containers, not your home directory. Apple Containers on Mac, namespaces on Linux.
scheduled Skills run on cron via your OS — launchd, systemd, whatever. No daemon sitting around.
memory Persistent context across tasks. Skills declare what they remember. Syncs across machines.
local-first Your machine, your keys, your data. Works offline. No account required.
$ wt "summarize my git log from this week" $ wt --skill compress --agent ollama $ wt schedule list $ wt thread # what happened today $ wt doctor # detect agents, keys, services
